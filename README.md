# Gestor de Tareas 📝  

Un gestor de tareas simple para organizar tu día a día.  

## 🚀 Tecnologías  
- Vue.js
- JavaScript 
- CSS

## 📥 Instalación  
1. Clona el repositorio:  

   git clone https://github.com/JoseSanchez90/To-Do-List.git

2. Instala dependencias:

   npm install

3. Ejecuta el Proyecto:

   npm run dev
